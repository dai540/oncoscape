CLI Reference
=============

Available console entry points:

- ``oncoscape-fetch-public``
- ``oncoscape-preflight``
- ``oncoscape-register``
- ``oncoscape-build-reference``
- ``oncoscape-make-teachers``
- ``oncoscape-extract-patches``
- ``oncoscape-train-breast``
- ``oncoscape-eval-render``
- ``oncoscape-report``
- ``oncoscape-run``

All commands accept:

- ``--config``
- ``--dry-run``

Scripts under ``scripts/`` are thin wrappers around the same CLI handlers and are suitable for direct use on clusters.
